Object Fit, Position, and View Box

This demo combines all `object-fit` modes with keyword, percentage, and four-value `object-position` placements. The last row also shows `object-view-box` crops using the three rectangle syntaxes.

`object-view-box` is only exposed by Chromium-class browsers today. The demo still loads elsewhere, but those cards will fall back to the uncropped image when the browser does not support the property.
Fill
object-fit: fill
Stretches the source to the frame.
None + Four-Value Offset
object-fit: none
object-position: right 18px bottom 12px
Contain + Keywords
object-fit: contain
object-position: left top
Cover + Percentages
object-fit: cover
object-position: 18% 78%
Scale-Down + Offset
object-fit: scale-down
object-position: right 12px bottom 10px
View Box via inset()
Crops the intrinsic image before fitting it into the frame.
View Box via xywh()
Uses an explicit source rectangle and then applies cover cropping.
View Box via rect()
Demonstrates the top/right/bottom/left rectangle form.
