Iframe Walking

Embedded documents can now join the export.

This demo mixes a same-origin iframe that loads another demo file with a rotated iframe rendered from srcdoc content. Enable walkIframes to traverse those nested documents and keep them in the same paint-order pipeline.

Embedded existing demo

The left frame points at text.html, so this page exercises same-origin iframe walking across a nested file load instead of only inline markup.

Transformed iframe

The right frame is rotated slightly so iframe descendants have to inherit the iframe viewport transform instead of using plain viewport-relative coordinates.
