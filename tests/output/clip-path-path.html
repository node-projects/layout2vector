Complex Clip Paths with path()

This demo covers the previously unsupported `clip-path: path(...)` cases: curved paths, compound evenodd windows, and image crops defined by SVG-style path data instead of the basic inset/circle/ellipse/polygon functions.

Curved Capsule
A single curved path clipped from a gradient box.
Evenodd Window
A compound `path()` clip with an interior cut-out.
Organic Photo Crop
An image clipped by a curved leaf-like silhouette.
Ticket Notches
A compound path with side cut-outs using the evenodd rule.
